Automatic Transfer Switch (ATS)

Mimic Diagram Visuel - Trois Positions

Position 1 – Utility

La charge est alimentée par le réseau principal. Le groupe électrogène est en veille.

Position 2 – Generator

Panne du réseau détectée → Transfert automatique vers le groupe électrogène.

Position 3 – Test / Neutral

Mode test ou maintenance : aucune source n’alimente la charge (position ouverte).
