Projet : Dashboard Unifié pour Système ESS et Domotique

📋 Description Générale

Développement d'une application web centralisée permettant à la fois :

· Le monitoring en temps réel d'un système de stockage d'énergie (ESS)
· Le contrôle et la commande des équipements domotiques et énergétiques

L'application intègre des visualisations Grafana (open source) et des composants de contrôle développés maison, le tout communiquant avec une base InfluxDB (open source) pour les données temporelles.

🧱 Stack Open Source Complète

Frontend

Composant Technologie Rôle
Framework React + TypeScript Interface utilisateur
Build tool Vite Compilation et développement
État global Zustand Gestion d'état légère
Requêtes API TanStack Query Cache et synchro données serveur
UI Components Shadcn/ui + TailwindCSS Composants accessibles et personnalisables
Graphiques (commande) Recharts ou ECharts Visualisations légères pour contrôles
MQTT WebSocket Temps réel bidirectionnel
Routing React Router Navigation

Backend / API

Composant Technologie Rôle
Serveur API Rust au lieu de Node.js + Express Endpoints REST pour commandes
MQTT WebSocket Communication temps réel
Proxy Grafana http-proxy-middleware Intégration sécurisée Grafana

Base de Données & Monitoring

Composant Technologie Rôle
Base temps réel InfluxDB OSS Stockage métriques ESS (température, SOC, puissance)
Visualisation Grafana OSS Dashboards monitoring
Reverse proxy Nginx Routage unifié (optionnel)

---

✅ Fonctionnalités Clés

Monitoring (via Grafana)

· Courbes de charge/décharge ESS
· État de charge (SOC) et état de santé (SOH)
· État de cellules
· Températures batterie et onduleur
· Historique de consommation/production
· Alertes configurées dans Grafana

Contrôle (composants maison)

· Limitation de puissance de décharge
· Mode de fonctionnement (auto, grid)
· Activation/Désactivation des équipements domotiques
· Scénarios programmables
· Audit trail des commandes (stocké dans InfluxDB)

Expérience Utilisateur

· Interface unifiée (pas de basculement entre outils)
· Mode kiosk Grafana (UI Grafana masquée)
· Thème clair/sombre synchronisé
· Layout draggable et persistant
· Responsive (desktop d'abord, tablette supportée)
